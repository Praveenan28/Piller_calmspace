# Piller

Piller is a sensory‑safe web application designed to provide children with a calm, predictable digital environment. It allows parents to configure comfort preferences and offers children simple, soothing micro‑activities such as counting, emotion selection, and guided breathing.

---

## Features

- Parent onboarding to configure a child’s comfort profile
- Child‑friendly home screen with calming visuals
- Interactive micro‑activities:
  - Garden world (gentle counting)
  - Feelings clouds (emotion selection)
  - Breathing bubbles (guided breathing)
- Profile persistence using Supabase
- No authentication required for MVP usage

---

## Tech Stack

- **Frontend:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Inline styles / CSS modules
- **Backend:** Supabase (PostgreSQL + API)
- **State Management:** React Context
- **Deployment Ready:** GitHub + Vercel compatible
